# Assignment-5

## Description
This repository contains two Python programs demonstrating the use of dictionaries and list slicing.

### 1. Task1.py
- Creates a dictionary of student names and marks
- Takes a student's name as input
- Displays the corresponding marks
- Shows a message if the student is not found

### 2. Task2.py
- Creates a list of numbers from 1 to 10
- Extracts the first five elements using slicing
- Reverses the extracted list
- Displays both lists

#### How to Run
1. Ensure Python is installed on your system.
2. Open a terminal or command prompt.
3. Run the programs
